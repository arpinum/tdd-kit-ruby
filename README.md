# TDD Kit - Ruby BDD

## Initialization

    rvm install 2.2.1
    bundle install

## Run unit tests

    rake test

## Run lint, unit tests and coverage

    rake

## Libraries documentation

* Assertion api: [minitest expectations]
* Linter: [rubocop]
* Cover: [SimpleCov]
* Mock api: [mocha]
